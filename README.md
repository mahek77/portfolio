# Mehek Mukesh Portfolio

A modern, minimal, single-page portfolio website for **Mehek Mukesh**, a BS Computer Science student, UI/UX designer, and web/Java developer based in Karachi, Pakistan.

This portfolio presents personal introduction, skills, projects, certifications, education, and contact details in a clean dark theme using a custom color palette.

## Color Palette

```css
--black: #000000ff;
--ink-black: #0c1821ff;
--prussian-blue: #1b2a41ff;
--charcoal-blue: #324a5fff;
--lavender: #ccc9dcff;
```

## Features

* Modern minimal dark portfolio design
* Responsive layout for desktop and mobile screens
* Smooth scrolling navigation
* Hero section with professional introduction
* About section with quick profile highlights
* Skills section for design, programming, database, and tools
* Project cards with live demo links where available
* Certifications and education sections
* Contact section with email, phone, location, and LinkedIn
* Built with HTML, CSS, and JavaScript

## How to Run the Project

No installation is required.

Simply open the HTML file in any modern web browser.

Steps:

1. Download the portfolio file.
2. Open the file in Chrome, Edge, Firefox, or any browser.
3. The website will run locally.

For example:
open mehek-portfolio-new-minimal-theme.html
Or simply double-click the HTML file.
